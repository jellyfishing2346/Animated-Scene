Multi-Layer Wick Editor Animation

This animation demonstrates a multi-layer scene with:

  • Background layer: Animated clouds, sun, and mountains
  • Middle layer: Moving cars on a road with trees
  • Foreground layer: Flying birds and falling leaves

The animation is set to 30 FPS and lasts for 4 seconds, designed to loop seamlessly.

Download Wick Project File

How to Use This Project

  1. Download the .wick file above
  2. Open it in Wick Editor (https://www.wickeditor.com/)
  3. You can modify any elements or animations
  4. To export your own version, go to File > Export > Animation/Video
